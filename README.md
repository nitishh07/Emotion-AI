# 🎭 Emotion AI - Facial Expression Detection

An AI-based project that detects human emotions from facial images using deep learning and image processing techniques.

---

## 🚀 Features
- Detects emotions from facial images
- Supports multiple expressions (Happy, Sad, Angry, etc.)
- Image preprocessing and augmentation
- Easy-to-use interface (upload image & get result)

---

## 🧠 Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Flask (for backend API)
- HTML/CSS/JavaScript (for frontend)
