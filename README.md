## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/nitishh07/emotion-ai.git
cd emotion-ai
```

### 2️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 3️⃣ Run backend server

```
python backend/app.py
```

### 4️⃣ Open frontend

* Open `frontend/index.html` in browser
* Upload image → Get emotion prediction

---

## 🧪 How It Works

1. User uploads an image
2. Image is sent to Flask backend
3. Image is preprocessed using OpenCV
4. Model predicts emotion
5. Result is returned and displayed
